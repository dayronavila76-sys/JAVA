# JAVA

Prácticas Java (migrado desde NetBeans). **JDK 21**.

## Ejecutar en terminal (Cursor)

Desde la carpeta `src`:

```powershell
cd C:\Users\dairo\Documents\JAVA\src
javac JavaApplication2.java
java JavaApplication2
```

Otra clase con `main` (por ejemplo `newvclass`):

```powershell
javac newvclass.java
java newvclass
```

En Cursor instala **Extension Pack for Java** si quieres ejecutar desde el editor (Run sobre `main`).
